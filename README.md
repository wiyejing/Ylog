# Ylog

Ylog 是一个基于 CodeIgniter V3.0.4的php博客系统。这是我学习php的第一个项目，也是第一次尝试使用github来管理代码。
如果您对我的项目有什么建议或指导，可联系我进一步交流。

# 致谢

- 后端UI基于 jquery-easyui  V1.4.4
- 框架基于 CodeIgniter  V3.0.4
- 编辑器采用 kindeditor V4.1.10

# 功能（已实现）

- 文档管理
- 数据库管理
- 回收站
- 第三方登录（QQ、微博、百度、人人……）
